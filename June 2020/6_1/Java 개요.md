## JDK 설치

- [JDK 1.8 설치 - java.sun.com](java.sun.com)

- 환경변수 설정
  - 시스템 변수에 추가 
    - 변수이름 JAVA_HOME
    - 변수 값 JDK가 설치된 디렉토리 입력
  - 시스템 변수안에 Path에 %JAVA_HOME%\bin 추가
- eclipse.org 설치



- "javac.exe" 파일의 역할
  - 자바언어로 작성 --> 컴퓨터에게 기계어로 번역하게 하기
  - 컴파일러  

- cmd 확인
  - java -version -> 버전 나옴
  - javac --> 명령어 나옴
  - java 환경 구축 성공
- 컴파일 및 실행명령어
  - 컴파일 명령어 : javac HelloWorld.java
  - 실행 명령어 : java Helloworld
