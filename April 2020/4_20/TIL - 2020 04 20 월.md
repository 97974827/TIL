- map 
  - 리스트의 요소를 지정된 함수로 처리해주는 함수
  - list(map(함수, 리스트))
  - 출처: https://bluese05.tistory.com/58 [ㅍㅍㅋㄷ]

```python
def func(x):
	return x * 2

>>> map( func, [1, 2, 3, 4] )
[2, 4, 6, 8]

>>> x = { 1 : 10, 2 : 20, 3: 30 }
>>> map(func, x)
[ 2, 4, 6 ]


>>> x = { 1 : 10, 2 : 20, 3: 30 }
>>> map(func, [ x[i] for i in x ])
[ 20, 40, 60 ]

```
